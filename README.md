# Peer-to-peer сетевой чат (консольный)

Учебный проект: двусторонний обмен текстовыми сообщениями между двумя участниками **напрямую по сети** (без отдельного сервера сообщений), с консольным интерфейсом.

## Возможности

- Режим **ожидания подключения:** процесс слушает TCP-порт и принимает одного собеседника.
- Режим **подключения:** процесс подключается к указанному хосту и порту peer-а.
- При запуске задаётся **имя пользователя**; для входящих сообщений в консоли отображаются **имя отправителя**, **время отправки** и **текст**.
